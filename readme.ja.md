## はじめに

### Logseq マーケットプレイスからインストール

- 右上のツールバーで [`---`] をクリックして [`プラグイン`] を開きます。マーケットプレイスを選択します。検索フィールドに `Show page date` と入力し、検索結果から選択してインストールします。

### 使用方法

- デフォルトでは、表示がオンになっています。
   > プラグイン設定で、日付をそのまま表示するか、ボタンを押して表示するかを選択できます。
