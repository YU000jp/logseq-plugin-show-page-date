## Getting Started

1. Install from Logseq Marketplace
   - Click [`---`] button on the top right toolbar
   - Select [`Plugins`]
   - Search for `Show page date`
   - Click Install

2. Configuration
   - Go to plugin settings
   - Choose your preferred display mode:
     - Always show dates
     - Show dates only when button is clicked
